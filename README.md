
# Uber Power BI Dashboard

An interactive Power BI dashboard created to analyze Uber ride bookings, revenue patterns, vehicle performance, ride outcomes, and customer/driver rating behavior. This project transforms raw ride data into meaningful business insights using data modeling, DAX measures, and visual analytics.
